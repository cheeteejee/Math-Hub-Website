Diameter Formulas

Different formulas can be used depending on the information that is given.

Main Formula

d = 2r

Use this formula when the radius is known.

Using Circumference

d = C ÷ π

Using Area

d = 2√(A ÷ π)

Relationship

r = d ÷ 2
